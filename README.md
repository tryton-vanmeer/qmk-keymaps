# QMK Keymaps

Keymaps for keyboards using [QMK Firmware](https://github.com/qmk/qmk_firmware)

## Massdrop Alt

### Layer 0

| Key | Notes      |
|-----|------------|
| FN  | layer 1    |
| CMD | layer 2    |

![layer0](assets/massdrop-alt/layer0.png)

### Layer 1

| Key  | Notes                                 |
|------|---------------------------------------|
| BOOT | put into bootloader mode for flashing |
